2)what is devops?

* Devops is a set of practices that combines software development and IT operations.
* It aims to shorten the system development life cycle and provide continuous delivery with hight software quality.
* Devops is complementary with agile software development ; several devops aspects came from the agile methodology.

9) what is hostname how can you find it?

* From the Start menu, select All Programs or Programs, then Accessories, and then Command Prompt.
* In the window that opens, at the prompt, enter hostname . 
* The result on the next line of the command prompt window will display the hostname of the machine without the domain.
